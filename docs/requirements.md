# SmartCS - 需求文档

## 1. 项目概述

SmartCS 是基于 LangGraph + FastAPI 构建的企业级多Agent智能客服系统。通过智能意图路由、专家Agent协作和人工介入升级，实现客服流程自动化。

## 2. 项目目标

- 客服响应时间降低 80% 以上
- 70% 以上的常规咨询实现自动化（账单、FAQ、订单状态）
- 复杂场景下实现无缝的人机协作
- 支持多模型LLM路由，优化成本与质量
- 提供完整的Agent决策可观测性

## 3. 用户角色

### 3.1 终端客户（聊天用户）
- 通过聊天界面提交服务咨询
- 接收专家Agent的自动回复
- 可随时请求人工客服

### 3.2 人工客服（坐席）
- 监控活跃会话
- 在AI升级时接管会话
- 可审查AI推理过程并覆盖决策

### 3.3 系统管理员
- 管理知识库（FAQ、产品文档）
- 配置Agent行为和路由规则
- 查看分析看板（解决率、响应时间等）

## 4. 功能需求

### 4.1 意图分类与路由
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-01 | 系统将用户意图分类为：账单、技术、退款、通用、人工升级 | P0 |
| FR-02 | 返回路由置信度分数，低置信度触发人工升级 | P0 |
| FR-03 | 支持可配置的意图类别（非硬编码） | P1 |

### 4.2 专家Agent
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-04 | BillingAgent：发票查询、支付历史、账单问题 | P0 |
| FR-05 | TechnicalAgent：产品故障排查、FAQ搜索、知识库检索 | P0 |
| FR-06 | RefundAgent：订单查询、退款资格检查、退款处理（需审批） | P0 |
| FR-07 | GeneralAgent：兜底处理、通用产品信息 | P1 |
| FR-08 | 每个Agent在会话内维护对话上下文 | P0 |

### 4.3 知识库（RAG）
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-09 | 支持文档上传（PDF、Markdown、TXT） | P0 |
| FR-10 | 基于向量的语义检索 | P0 |
| FR-11 | 知识库增删改查管理接口 | P1 |
| FR-12 | 增量更新，无需全量重建索引 | P2 |

### 4.4 人工介入（Human-in-the-Loop）
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-13 | Agent执行可暂停等待人工审批（如退款超过一定金额） | P0 |
| FR-14 | 人工客服可随时接管会话 | P0 |
| FR-15 | 人工介入后，释放会话可交还AI继续处理 | P1 |

### 4.5 会话管理
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-16 | 完整会话历史存储和查询 | P0 |
| FR-17 | 会话状态持久化（系统重启后可恢复） | P0 |
| FR-18 | 支持多轮对话并保持上下文 | P0 |

### 4.6 多模型支持
| ID | 需求 | 优先级 |
|----|------|--------|
| FR-19 | 支持 OpenAI、Anthropic、本地模型后端 | P0 |
| FR-20 | 简单查询路由到便宜模型，复杂查询路由到强模型 | P1 |
| FR-21 | 每个Agent可独立配置模型（如Router用快模型，Refund用强模型） | P1 |

## 5. 非功能需求

| ID | 需求 | 目标 |
|----|------|------|
| NFR-01 | API响应延迟（P95） | < 5秒（非流式） |
| NFR-02 | 流式响应首token延迟 | < 1秒 |
| NFR-03 | 并发会话数 | 100+ |
| NFR-04 | 系统可用性 | 99.5% |
| NFR-05 | 所有LLM调用带token用量追踪 | 100%覆盖 |
| NFR-06 | 日志中PII检测与脱敏 | 必须 |
| NFR-07 | API认证 | JWT |

## 6. 用户故事

### 终端客户
1. **US-01**：我想咨询账单问题，不需要等待人工就能得到即时回复
2. **US-02**：我想描述技术问题，让Agent自动检索知识库找到解决方案
3. **US-03**：我想申请退款，让Agent自动检查我的退款资格
4. **US-04**：当AI无法帮我时，我想转接人工客服
5. **US-05**：我希望Agent记住我们的对话上下文，不要让我重复描述

### 人工客服
6. **US-06**：我想看到哪些会话需要我关注（被AI升级的）
7. **US-07**：我想看到AI在无法解决问题时的推理过程
8. **US-08**：我想接管会话后AI完全退出

### 系统管理员
9. **US-09**：我想上传产品文档到知识库
10. **US-10**：我想查看指标：解决率、平均响应时间、Agent性能
11. **US-11**：我想配置每个Agent使用的LLM模型

## 7. v1 不在范围内

- 多语言支持
- 语音输入
- 移动端原生App
- 社交媒体集成（微信、WhatsApp）
- 情感分析
- 主动外呼
